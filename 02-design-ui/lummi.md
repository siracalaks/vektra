# Lummi

**What it does:** AI image library and generator with royalty-free assets and Figma/Canva/Webflow plugins.

**Stage / workflow:** Asset sourcing — find or generate royalty-free imagery for designs.

**Cost model:** Freemium — free downloads with limits; paid Pro for unlimited and higher-res assets.

**Connection to Claude/n8n:** Web app plus design-tool plugins; not a general code API.

**Connection recipe:**
```bash
# Install from Figma Community: Lummi (also Canva / Webflow plugins)
# Or browse https://lummi.ai
```

**When to use it:** When you need royalty-free, on-trend imagery or AI-generated visuals for UI and marketing.

**Why it's useful:** Supplies hero/marketing imagery for the Next.js 15 app; plugins fit a Figma-based design flow.

**Notable alternatives:** Unsplash, Pexels, Freepik.
