# Graphic Design Prompt Pack

**What it does:** A set of ~6 reusable Claude prompts for graphic-design work — these are prompts, not a product.

**Stage / workflow:** Design ideation — run them directly in Claude to brief, generate, and critique design work.

**Cost model:** Free — they are prompts you paste into Claude.

**Connection to Claude/n8n:** Use as prompts/skills inside Claude; no install.

**Connection recipe:**
```bash
# No install — use these 6 prompts directly with Claude:
# 1. Design brief simplifier — distill a messy brief into clear goals, audience, constraints, deliverables.
# 2. Logo concept generator — produce multiple distinct logo directions with rationale.
# 3. Layout & composition builder — propose grid, hierarchy, and spacing for a given canvas/format.
# 4. Typography system generator — pick type pairings, scale, weights, and usage rules.
# 5. Design critique & improvement — review a design and give specific, prioritized fixes.
# 6. Brand identity system builder — assemble color, type, voice, and usage into a coherent identity.
# (Exact source pack unverified — these are documented prompt ideas, not an official product.)
```

**When to use it:** When you want structured, repeatable Claude assistance for briefs, logos, layout, type, critique, or brand systems.

**Why it's useful:** Turns Claude into a design assistant that feeds visual direction and brand decisions into the Next.js 15 + shadcn/Tailwind UI.

**Notable alternatives:** Custom Claude project instructions, design system docs, Figma AI prompts.
