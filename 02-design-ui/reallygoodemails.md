# Really Good Emails

**What it does:** Gallery of 19,000+ email design examples, many with viewable code.

**Stage / workflow:** Inspiration — reference email layouts, patterns, and HTML structure.

**Cost model:** Freemium — free browsing; paid membership for advanced search/collections.

**Connection to Claude/n8n:** Browse-only website; no public API.

**Connection recipe:**
```bash
# Browse / bookmark: https://reallygoodemails.com
```

**When to use it:** When designing transactional or marketing emails and you need proven layout/code patterns.

**Why it's useful:** Email patterns inform templates for the product's transactional/marketing emails (e.g. via n8n flows).

**Notable alternatives:** Email Love, Milled, Mailcharts.
