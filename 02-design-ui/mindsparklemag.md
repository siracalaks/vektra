# Mindsparkle Mag

**What it does:** Design inspiration gallery covering branding, web, graphic, and packaging design.

**Stage / workflow:** Inspiration — browse curated work to inform visual direction.

**Cost model:** Free to browse.

**Connection to Claude/n8n:** Browse-only website; no API.

**Connection recipe:**
```bash
# Browse / bookmark: https://mindsparklemag.com
```

**When to use it:** When you need a mood-board or visual direction reference before designing.

**Why it's useful:** Source of high-quality visual references to shape the look of the Next.js 15 + shadcn/Tailwind UI.

**Notable alternatives:** Awwwards, Behance, Dribbble.
