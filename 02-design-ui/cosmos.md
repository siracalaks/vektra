# Cosmos

**What it does:** AI-powered visual inspiration app to search, save, and organize images into boards.

**Stage / workflow:** Inspiration — collect and curate visual references for projects.

**Cost model:** Free to browse and save (account required).

**Connection to Claude/n8n:** Browse-only app; no public API.

**Connection recipe:**
```bash
# Browse / bookmark: https://cosmos.so
```

**When to use it:** When you want to build curated visual mood boards and discover related references.

**Why it's useful:** Helps assemble a visual language reference that informs the Next.js 15 + shadcn/Tailwind UI direction.

**Notable alternatives:** Pinterest, Are.na, Savee.
