# Photopea

**What it does:** Free browser-based Photoshop alternative for raster and vector image editing (supports PSD).

**Stage / workflow:** Asset editing — edit images, mockups, and exported design assets without installing software.

**Cost model:** Freemium — fully usable free (ad-supported); paid tier removes ads.

**Connection to Claude/n8n:** Web app; offers a scripting/embed API for advanced automation.

**Connection recipe:**
```bash
# Web app — open https://photopea.com (no signup needed)
# An embed/scripting API exists for automation — see Photopea API docs.
```

**When to use it:** When you need quick PSD-compatible image edits or asset tweaks without Photoshop.

**Why it's useful:** Free way to prep and edit images/assets that feed into the Next.js 15 app's UI and content.

**Notable alternatives:** GIMP, Pixlr, Figma (for vector).
