# FlowMapp

**What it does:** Visual UX planning tool for sitemaps, user flows, and wireframes.

**Stage / workflow:** UX architecture — plan information architecture and flows before UI design.

**Cost model:** Freemium — free tier with limited projects; paid plans for more projects/collaboration.

**Connection to Claude/n8n:** Web app; no documented public API.

**Connection recipe:**
```bash
# Web app — sign up at https://flowmapp.com
# No install / no public API unless noted.
```

**When to use it:** When you need to map a site's structure and user journeys before building screens.

**Why it's useful:** Sitemaps and flows map cleanly to Next.js 15 route structure (app router) and page planning.

**Notable alternatives:** Whimsical, Octopus.do, FigJam.
