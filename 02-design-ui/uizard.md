# Uizard

**What it does:** Turns hand sketches, screenshots, or text prompts into editable, multi-screen UI mockups.

**Stage / workflow:** Early design — fast wireframe and mockup generation before detailed UI work.

**Cost model:** Freemium — free tier with limited projects; paid Pro/Business tiers unlock unlimited projects and AI features.

**Connection to Claude/n8n:** Web app; no public AI generation API in the open at time of writing.

**Connection recipe:**
```bash
# Web app — sign up at https://uizard.io
# No install / no public API unless noted.
```

**When to use it:** When you have a napkin sketch or rough idea and want a clickable mockup in minutes to validate a concept.

**Why it's useful:** Produces quick visual mockups you can translate into Next.js 15 + shadcn/Tailwind components, useful at the ideation-to-design handoff.

**Notable alternatives:** Vercel v0, Visily, Banani.
