# The Dieline

**What it does:** Packaging-design inspiration gallery with 20,000+ curated projects.

**Stage / workflow:** Inspiration — reference packaging and brand visual work.

**Cost model:** Free to browse.

**Connection to Claude/n8n:** Browse-only website; no API.

**Connection recipe:**
```bash
# Browse / bookmark: https://thedieline.com
```

**When to use it:** When a project touches physical product/packaging branding and you need visual references.

**Why it's useful:** Useful brand/packaging references when product work extends beyond the digital Next.js app.

**Notable alternatives:** Packaging of the World, Behance, Mindsparkle Mag.
