# Leonxlnx/taste-skill

**What it does:** Gives your AI good taste and stops generic slop, tuned for landing pages, portfolios, and redesigns via VARIANCE / MOTION / DENSITY dials. Not intended for dashboards.

**Stage / workflow:** Design & UI stage — applied to marketing/landing/portfolio surfaces to push output away from templated AI aesthetics.

**Cost model:** Free, open-source skill.

**Connection to Claude/n8n:** Claude Code skill.

**Connection recipe:**
```bash
npx skills add https://github.com/Leonxlnx/taste-skill
```

**When to use it:** When designing a landing page, portfolio, or redesign and you want a distinctive look — adjust the VARIANCE/MOTION/DENSITY dials to taste.

**Why it's useful:** Produces high-taste landing/portfolio UI that ships as Next.js 15 + Tailwind pages; complements shadcn/ui for marketing surfaces.

**Notable alternatives:** impeccable, emilkowalski/skill, anthropics/skills (frontend-design).
