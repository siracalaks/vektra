# Gamma

**What it does:** AI generator for presentations, decks, documents, and simple websites from a prompt.

**Stage / workflow:** Content/deck creation — go from outline or prompt to a styled deck or page.

**Cost model:** Freemium — free credits; paid Plus/Pro for unlimited AI and advanced export.

**Connection to Claude/n8n:** Web app; Gamma offers a generate API (verify current availability/scope).

**Connection recipe:**
```bash
# Web app — sign up at https://gamma.app
# A Gamma generate API exists on paid plans — check https://gamma.app for current docs.
```

**When to use it:** When you need a good-looking pitch deck or one-pager fast without manual slide design.

**Why it's useful:** Quick way to produce pitch/marketing decks for the product without leaving the design stage.

**Notable alternatives:** Presentations.ai, Pitch, Beautiful.ai.
