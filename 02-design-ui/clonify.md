# Clonify

**What it does:** Framer/Figma UI library with 1000+ responsive, ready-to-use sections and templates.

**Stage / workflow:** Design assembly — drop in pre-built sections to compose pages quickly.

**Cost model:** Freemium — free components with paid full library access (verify current pricing).

**Connection to Claude/n8n:** Figma/Framer plugin and asset library; not a code API.

**Connection recipe:**
```bash
# Install from Figma Community: Clonify
# Also usable in Framer — see https://clonify.io
```

**When to use it:** When you want to assemble polished marketing pages fast from proven section patterns.

**Why it's useful:** Section patterns translate into shadcn/Tailwind blocks for the Next.js 15 app, speeding up landing pages.

**Notable alternatives:** Tailwind UI, shadcn/ui blocks, Framer templates.
