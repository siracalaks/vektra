# Figma AI

**What it does:** Native AI features inside Figma — prompt-to-app (Figma Make), visual design search, and text generation.

**Stage / workflow:** Core design — accelerate work directly within the Figma canvas.

**Cost model:** Freemium — included with Figma plans; some AI features metered or tier-gated.

**Connection to Claude/n8n:** Built into the Figma app; access via Figma plugins/community, not a standalone API.

**Connection recipe:**
```bash
# Use inside Figma — enable AI features in your Figma account.
# Figma Make and AI tools live in the Figma app: https://figma.com/ai
```

**When to use it:** When you already design in Figma and want AI assists without leaving the canvas.

**Why it's useful:** Figma is the common design source for handoff; exports and Dev Mode help translate to Next.js 15 + shadcn/Tailwind.

**Notable alternatives:** Penpot, Google Stitch, Uizard.
