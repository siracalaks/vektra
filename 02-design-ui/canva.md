# Canva

**What it does:** All-in-one design platform for graphics, presentations, and social content with AI tools and a developer API.

**Stage / workflow:** Visual content — produce branded graphics, social posts, and decks quickly.

**Cost model:** Freemium — free tier plus Canva Pro for premium assets, brand kit, and more AI credits.

**Connection to Claude/n8n:** Web app with a Canva Connect API and app platform for integrations/automation.

**Connection recipe:**
```bash
# Web app — sign up at https://canva.com
# Canva Connect API exists for integrations — see https://www.canva.dev for current docs.
```

**When to use it:** When you need fast branded marketing/social graphics rather than full UI design.

**Why it's useful:** Produces marketing assets and social content around the Next.js 15 app; the API enables n8n automation.

**Notable alternatives:** Figma, Adobe Express, Pixlr.
