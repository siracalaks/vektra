# tldraw/tldraw

**What it does:** An npm infinite-canvas SDK for building whiteboard and diagram UIs in React.

**Stage / workflow:** Design & UI stage — embeds an interactive canvas for whiteboarding, diagramming, and visual editing features inside an app.

**Cost model:** Free, open-source SDK (review tldraw's license terms for production/commercial use).

**Connection to Claude/n8n:** npm package imported into your React/Next.js app.

**Connection recipe:**
```bash
npm install tldraw
```

**When to use it:** When you need an embeddable infinite canvas — whiteboards, diagram editors, or visual planning surfaces — without building canvas internals.

**Why it's useful:** Drops straight into Next.js 15 as a React component, styleable alongside Tailwind and shadcn/ui.

**Notable alternatives:** Excalidraw, React Flow, Konva.
