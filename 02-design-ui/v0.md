# Vercel v0

**What it does:** Generates React/Next.js UI with shadcn/ui and Tailwind from text prompts or screenshots.

**Stage / workflow:** Design-to-code — produce production-leaning component code directly from a description or reference image.

**Cost model:** Freemium — free monthly credits; paid Premium/Team plans for more credits and an API.

**Connection to Claude/n8n:** Web app plus a v0 Platform API; integrates naturally with Vercel deploys.

**Connection recipe:**
```bash
# Web app — sign up at https://v0.dev
# A v0 Platform API exists for paid plans; check https://v0.dev for current API docs.
```

**When to use it:** When you want a head start on real Next.js + shadcn components rather than just a static mockup.

**Why it's useful:** Native fit — output is exactly Next.js 15 + shadcn/Tailwind, droppable into the existing project with minimal rework.

**Notable alternatives:** Uizard, Google Stitch, bolt.new.
