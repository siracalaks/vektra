# Presentations.ai

**What it does:** AI deck generator that builds presentations from text or an outline.

**Stage / workflow:** Deck creation — turn raw text into a structured, styled presentation.

**Cost model:** Freemium — free tier with paid upgrades for more decks and features.

**Connection to Claude/n8n:** Web app; no documented public API.

**Connection recipe:**
```bash
# Web app — sign up at https://presentations.ai
# No install / no public API unless noted.
```

**When to use it:** When you want an automated deck from prose and prefer a guided, template-driven flow.

**Why it's useful:** Fast deck output for product/marketing communication alongside the Next.js 15 app.

**Notable alternatives:** Gamma, Beautiful.ai, Pitch.
