# Edits (Instagram)

**What it does:** Instagram's native mobile video-editing app — Reels-oriented editing with captions, effects, and creator/insights tools.

**Stage / workflow:** Manual post-production for vertical social. Finish and publish 9:16 clips directly into the Instagram/Reels ecosystem.

**Cost model:** Free (Meta app).

**Connection to Claude/n8n:** Reference — no public automation API. Mobile app, manual editing only.

**Connection recipe:**
```bash
# Mobile app — no public API; edit and publish manually via the app.
```

**When to use it:** Editing and publishing Reels-native vertical content when staying inside Instagram's tooling.

**Why it's useful:** Purely manual; relevant as the publishing/finishing end for 9:16 content, not part of the automated flow.

**Notable alternatives:** CapCut (cross-platform), Clipchamp (web), Remotion (programmatic, in-stack).
