# Motion (Framer Motion)

**What it does:** Production-ready animation library for React (formerly Framer Motion), with declarative gestures, layout and spring animations.

**Stage / workflow:** Animation layer — page transitions, micro-interactions and gesture-driven motion across the UI.

**Cost model:** Free and open source.

**Connection to Claude/n8n:** npm package.

**Connection recipe:**
```bash
npm install motion
```

**When to use it:** Whenever you need expressive, physics-based animations and transitions in React components.

**Why it's useful:** First-class React/TypeScript support and the animation engine behind many shadcn-compatible component libraries.

**Notable alternatives:** GSAP, React Spring, Auto-Animate.
