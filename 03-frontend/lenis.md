# Lenis

**What it does:** Lightweight smooth-scroll library for the web.

**Stage / workflow:** Scroll-experience polish — adds buttery smooth scrolling and drives scroll-linked animations.

**Cost model:** Free and open source.

**Connection to Claude/n8n:** npm package.

**Connection recipe:**
```bash
npm install lenis
```

**When to use it:** On marketing or portfolio pages where smooth scroll and scroll-triggered motion elevate the feel.

**Why it's useful:** Framework-agnostic with a React wrapper, drops into Next.js and pairs well with Motion for scroll effects.

**Notable alternatives:** Locomotive Scroll, react-scroll, native CSS scroll-behavior.
