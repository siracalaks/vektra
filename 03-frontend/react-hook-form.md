# React Hook Form

**What it does:** Performant, uncontrolled-first form library for React with minimal re-renders.

**Stage / workflow:** Form handling layer — registration, validation and submission for any form-driven screen.

**Cost model:** Free and open source.

**Connection to Claude/n8n:** npm package.

**Connection recipe:**
```bash
npm install react-hook-form
```

**When to use it:** Any non-trivial form where you want fast performance and easy validation wiring.

**Why it's useful:** Pairs natively with Zod (resolver) and shadcn/ui's Form component in Next.js + TypeScript.

**Notable alternatives:** TanStack Form, Formik, Final Form.
