# React Bits

**What it does:** Collection of animated, interactive and customizable React components.

**Stage / workflow:** UI flourish layer — text animations, backgrounds and interactive elements added on top of your component base.

**Cost model:** Free and open source.

**Connection to Claude/n8n:** CLI generator via the shadcn registry.

**Connection recipe:**
```bash
npx shadcn@latest add @react-bits/<Component>
# verify exact component path
```

**When to use it:** When you want ready-made flashy animated components to differentiate a page quickly.

**Why it's useful:** shadcn-style install slots into Next.js + React + TypeScript + Tailwind with no extra tooling.

**Notable alternatives:** Magic UI, Aceternity UI, Motion Primitives.
