# build-your-own-x

**What it does:** A curated collection of "build your own X" tutorials (database, git, browser, shell, and more) that teach by reimplementing real tools from scratch.

**Stage / workflow:** Deep-learning-by-building stage — understand systems by recreating them.

**Cost model:** Free, open source (reference-only).

**Connection to Claude/n8n:** Reference — bookmark/mirror the repo; no install.

**Connection recipe:**
```bash
# Reference only — read online or mirror locally:
git clone https://github.com/codecrafters-io/build-your-own-x
```

**When to use it:** When you want to truly understand a technology by building a minimal version of it yourself.

**Why it's useful:** Building tools from scratch sharpens the systems intuition that makes integration and automation work less of a black box.

**Notable alternatives:** CodeCrafters challenges, project-based-learning, Crafting Interpreters.
