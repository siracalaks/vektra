# awesome-mcp-servers

**What it does:** The canonical curated awesome-list of Model Context Protocol (MCP) servers and the discovery source for new MCP servers.

**Stage / workflow:** MCP-discovery stage — the place to scout new MCP servers to add to this library.

**Cost model:** Free, open source (reference-only).

**Connection to Claude/n8n:** Reference — bookmark/mirror the repo; no install.

**Connection recipe:**
```bash
# Reference only — read online or mirror locally:
git clone https://github.com/punkpeye/awesome-mcp-servers
```

**When to use it:** When looking for new MCP servers to connect to Claude or to evaluate and catalog in this resource library.

**Why it's useful:** It is the primary feed for finding MCP servers that extend Claude's capabilities — the upstream source for this library's MCP entries.

**Notable alternatives:** modelcontextprotocol/servers (official), mcp.so directory, Smithery registry.
