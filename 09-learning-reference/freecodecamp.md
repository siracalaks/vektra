# freeCodeCamp

**What it does:** Free, full-stack web development curriculum with hands-on projects and verifiable certifications.

**Stage / workflow:** Foundational learning stage — structured coursework from basics through full-stack and data topics.

**Cost model:** Free, open source (reference-only).

**Connection to Claude/n8n:** Reference — bookmark/mirror the repo; no install.

**Connection recipe:**
```bash
# Reference only — read online or mirror locally:
git clone https://github.com/freeCodeCamp/freeCodeCamp
```

**When to use it:** When you want a guided, project-driven curriculum with certifications rather than scattered tutorials.

**Why it's useful:** Solid free baseline for web/full-stack fundamentals that underpin building on top of Claude and n8n.

**Notable alternatives:** The Odin Project, Codecademy, MDN Learn Web Development.
