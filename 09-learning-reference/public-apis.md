# public-apis

**What it does:** A large, categorized directory of free public APIs for use in projects.

**Stage / workflow:** Project-sourcing stage — find data and service endpoints to wire into builds and demos.

**Cost model:** Free, open source (reference-only).

**Connection to Claude/n8n:** Reference — bookmark/mirror the repo; no install.

**Connection recipe:**
```bash
# Reference only — read online or mirror locally:
git clone https://github.com/public-apis/public-apis
```

**When to use it:** When you need a data source or third-party service to power a side project, demo, or n8n workflow.

**Why it's useful:** n8n thrives on connecting APIs; this is a ready catalog of endpoints to automate against.

**Notable alternatives:** RapidAPI Hub, APIs.guru, ProgrammableWeb archives.
