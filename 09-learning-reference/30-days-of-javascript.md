# 30-Days-Of-JavaScript

**What it does:** A 30-day, step-by-step JavaScript challenge covering language fundamentals through DOM and beyond, with daily exercises.

**Stage / workflow:** Hands-on JS skill-building stage — a paced, exercise-led path to JavaScript fluency.

**Cost model:** Free, open source (reference-only).

**Connection to Claude/n8n:** Reference — bookmark/mirror the repo; no install.

**Connection recipe:**
```bash
# Reference only — read online or mirror locally:
git clone https://github.com/Asabeneh/30-Days-Of-JavaScript
```

**When to use it:** When you want a structured, time-boxed way to learn or refresh JavaScript through daily practice.

**Why it's useful:** JavaScript is the lingua franca for n8n Function nodes and scripting glue, so a focused refresher is directly useful.

**Notable alternatives:** Eloquent JavaScript, javascript.info, freeCodeCamp JS curriculum.
