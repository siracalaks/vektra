# project-based-learning

**What it does:** A curated list of project-based programming tutorials organized by language and technology.

**Stage / workflow:** Applied-practice stage — learn a language or framework by building a complete project.

**Cost model:** Free, open source (reference-only).

**Connection to Claude/n8n:** Reference — bookmark/mirror the repo; no install.

**Connection recipe:**
```bash
# Reference only — read online or mirror locally:
git clone https://github.com/practical-tutorials/project-based-learning
```

**When to use it:** When you want to learn a new language or stack through a guided, end-to-end build rather than isolated exercises.

**Why it's useful:** Project-first learning maps directly onto an A-to-Z "build something real" workflow.

**Notable alternatives:** build-your-own-x, freeCodeCamp projects, The Odin Project.
