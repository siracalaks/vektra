# ai-agents-for-beginners

**What it does:** Microsoft's beginner-friendly curriculum (lessons + code) for building AI agents.

**Stage / workflow:** Agent-fundamentals learning stage — structured intro to designing and building AI agents.

**Cost model:** Free, open source (reference-only).

**Connection to Claude/n8n:** Reference — bookmark/mirror the repo; no install.

**Connection recipe:**
```bash
# Reference only — read online or mirror locally:
git clone https://github.com/microsoft/ai-agents-for-beginners
```

**When to use it:** When ramping up on agent concepts (tools, memory, planning) before building agents on Claude or in n8n.

**Why it's useful:** Directly relevant to building Claude-powered agents and orchestrating them through automation.
