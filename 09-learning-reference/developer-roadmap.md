# developer-roadmap

**What it does:** Community-curated visual roadmaps for becoming a developer across frontend, backend, devops, and many other tracks.

**Stage / workflow:** Skill-planning stage — pick a path and map the topics to learn before/while building.

**Cost model:** Free, open source (reference-only).

**Connection to Claude/n8n:** Reference — bookmark/mirror the repo; no install.

**Connection recipe:**
```bash
# Reference only — read online or mirror locally:
git clone https://github.com/kamranahmedse/developer-roadmap
```

**When to use it:** When deciding what to learn next or sanity-checking the breadth of a role's required skills.

**Why it's useful:** Gives a structured, opinionated learning path so reference reading stays goal-directed rather than ad hoc.

**Notable alternatives:** roadmap.sh (hosted version), Teclado roadmaps, Coursera/edX learning paths.
