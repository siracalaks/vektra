# awesome-python

**What it does:** A curated, categorized list of Python frameworks, libraries, tools, and resources.

**Stage / workflow:** Library-discovery stage — find the right Python package for a given task.

**Cost model:** Free, open source (reference-only).

**Connection to Claude/n8n:** Reference — bookmark/mirror the repo; no install.

**Connection recipe:**
```bash
# Reference only — read online or mirror locally:
git clone https://github.com/vinta/awesome-python
```

**When to use it:** When starting a Python project and you want a vetted shortlist of libraries per domain instead of searching blindly.

**Why it's useful:** Python powers many AI/automation glue scripts; this is a fast way to pick mature, well-known packages.

**Notable alternatives:** PyPI search, Awesome Python Applications, Python Weekly.
