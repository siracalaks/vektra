# claude-seo

**What it does:** Universal SEO skill for Claude Code with 25 sub-skills and 18 sub-agents covering technical, on-page, and content SEO.

**Stage / workflow:** Marketing/growth stage — keyword research, content optimization, technical audits, and SEO automation driven from Claude Code.

**Cost model:** Free, open-source. Requires Python 3.10+ on the host.

**Connection to Claude/n8n:** Skill — installed as a Claude Code plugin marketplace.

**Connection recipe:**
```bash
/plugin marketplace add AgriciDaniel/claude-seo
```

**When to use it:** When you need a deep, multi-agent SEO workflow (audits, content briefs, on-page fixes) rather than one-off SEO copy.

**Why it's useful:** Runs inside Claude Code and can audit/optimize the Next.js sites; outputs can be scheduled or actioned via n8n.

**Notable alternatives:** coreyhaines31/marketingskills (broader marketing), Surfer SEO, Ahrefs.
