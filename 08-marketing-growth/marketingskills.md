# marketingskills

**What it does:** Marketing skills for Claude Code and AI agents covering CRO, copywriting, SEO, analytics, and growth engineering.

**Stage / workflow:** Sits at the marketing/growth stage once a product exists — drafting copy, optimizing conversion, and planning SEO/analytics work directly inside Claude Code.

**Cost model:** Free, open-source skill marketplace.

**Connection to Claude/n8n:** Skill — installed into Claude Code as a plugin marketplace.

**Connection recipe:**
```bash
/plugin marketplace add coreyhaines31/marketingskills
```

**When to use it:** When you want marketing expertise (landing-page CRO, email/ad copy, SEO strategy) on tap inside Claude Code without leaving your editor.

**Why it's useful:** Plugs straight into Claude Code, so it can write copy and growth assets for the Next.js sites and feed campaigns orchestrated through n8n.

**Notable alternatives:** AgriciDaniel/claude-seo (SEO-focused skill), HubSpot AI tools, Jasper.
